Releasing webjars-play
----------------------

1. Tag the release: `git tag -s v2.9.0`
1. Push tag: `git push upstream v2.9.0`
1. GitHub action workflow does the rest: https://github.com/webjars/webjars-play/actions/workflows/publish.yml
